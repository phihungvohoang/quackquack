# Auto nhặt trứng, ấp trứng, ăn vịt vàng

### Cài node js tại (https://nodejs.org/en/download/prebuilt-installer)

### Run: git clone https://github.com/phihungvohoang/quackquack.git

### Run: cd quack

### Run: node index.js
